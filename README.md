# Excel 智能数据统计工具

## 📋 工具简介

> Excel 数据统计工具是一款轻量的桌面应用程序，专为需要处理多个 Excel 文件并进行简单的任务数据统计的用户设计。支持跨文件、跨工作表的灵活数据计算，提供可视化的规则构建器和智能筛选功能。

![Excel统计界面](Example.png)

### ✨ 核心特性

- 🎯 **多文件管理**：支持同时加载最多 6 个 Excel 文件，自动处理同名文件
- 📊 **跨表统计**：可在不同文件、不同工作表之间自由组合数据进行计算
- 🔧 **灵活规则**：可视化规则构建器，支持 AND/OR 逻辑组合
- 🎨 **多种计算模式**：单元格、整行、整列（范围）、自定义值
- 🔍 **智能筛选**：支持关键字筛选和列值筛选，可组合多个筛选条件
- 💾 **结果管理**：支持保存多个计算结果，可重命名和删除
- 🖥️ **现代界面**：基于 Electron + React + Ant Design 构建，界面美观易用

## 🚀 快速开始

### 安装运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm start

# 打包应用（Windows）
npm run package
```

### 系统要求

- Windows 10 或更高版本
- Node.js 16+ （开发环境）

- **语言**：TypeScript
